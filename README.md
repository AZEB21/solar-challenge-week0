# solar-challenge-week0

10 Academy Week 0 Solar Data Challenge

This repository contains my Week 0 project submission for the 10 Academy Solar Challenge.

## Contents

- README.md — project overview
- .gitignore — files to ignore
- requirements.txt — Python dependencies


## Project Overview
This project is part of the Solar Challenge Week 0. It focuses on generating datasets for multiple countries and analyzing them using Jupyter notebooks. The project includes scripts for dataset creation, exploratory data analysis (EDA), and country comparison.
